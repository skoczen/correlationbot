Find patterns in your data.

CorrelationBot is a simple way to find patterns in your data. It speaks HTTP, and has just one endpoint.

Using CorrelationBot

CorrelationBot speaks JSON over HTTP. That's it.

Python

import requests

requests.post("http://correlationbot.com", data={
    "data": [
        [1, 2, 3, 4],
        [5, 6, 7, 8]
    ]
})

print requests.json
>>> 
{
    "correlations": {
        "column_1": "1",
        "column_2": "2",
        "correlation": 0.93,
        "covariance": -0.93,
        "pearson": 0.93,
        "spearman": 0.4,
        "kendall": 0.2,
    },
    // ... 
]

Curl

$ curl -X POST -H "Content-Type: application/json" -d '{"data": [[1, 2, 3, 4],[5, 6, 7, 8]]}' http://correlationbot.com
